# ProjektProgramistyczny

OPIS:
Projekt przygotowany na zajęcia z Indywidualnego projektu programistycznego na drugim roku informatyki na UKSW.
Program ma na celu skanowanie sieci w poszukiwaniu niezabezpieczonych urządzeń IOTs, pobranie ich adresów IP oraz przygotowanie bazy takich urządzeń.
